Mamesaver

Mamesaver is a Windows screen saver which playes MAME ROMS for a specified time.

Features

  • Choose which ROMs get played
  • In-Game game information and customization
  • Multi-monitor support and cloning
  • In-Game hotkeys - jump in and play the game, move back and forth or just remove from the list if you don't wanna see it again
  • Splash screen with some customization features

Screen shots

Requirements

  • Microsoft .NET 4.6 Framework
  • A working setup of MAME
  • Some ROMS
